# Full Stack Trading Dashboard

A full-stack quantitative trading dashboard built with React and FastAPI that simulates algorithmic trading strategies, portfolio backtesting, and exchange-style order matching.

## Features

- Interactive stock market visualization
- Moving average crossover strategy backtesting
- Portfolio performance simulation
- Order book system
- Trade matching engine
- Cloudflare Turnstile bot protection
- REST API backend with FastAPI
- Responsive modern frontend with React

## Technologies Used

### Frontend
- React
- Vite
- Recharts
- JavaScript
- Tailwind CSS

### Backend
- FastAPI
- Python
- Pandas

### Infrastructure / APIs
- Cloudflare Turnstile

## Project Highlights

- Built a complete frontend + backend architecture
- Implemented moving average trading strategy simulation
- Created real-time order matching logic similar to exchange systems
- Integrated Cloudflare security verification for bot protection
- Processed and analyzed historical market data using Pandas
